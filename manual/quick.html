ครั้งแรกครั้งเดียว

  1. ติดตั้งและเปิดโปรแกรม — Mac: คลิกขวา → Open → Open · Windows: More info → Run anyway (เฉพาะครั้งแรก)
  2. แท็บ ตั้งค่า → วาง FAL API Key → กด "ทดสอบ" ให้ขึ้น ✓ · ตั้ง concurrency 30 · Quality เริ่มต้น Low

เจนภาพ 1 batch (แท็บ "สร้างภาพ")

  1. เลือกรูปหุ่น 1 รูป (JPG/PNG/HEIC จาก iPhone ได้เลย — ratio ผลลัพธ์เท่ารูปหุ่น)
  2. เลือกโฟลเดอร์เสื้อผ้า — คลิกรูปเพื่อติ๊กออกได้ · คู่หน้า/หลัง: ตั้งชื่อ _front/_back จับคู่ให้เอง
  3. เลือกมุม Front / Back / Side (เลือกหลายมุมได้ · 1 มุม = 1 รูป)
  4. เลือก Quality + โฟลเดอร์เซฟผล + Prompt 1/2
  5. ดูราคาที่แถบล่าง → กด ▶ เริ่มเจนทั้งหมด
