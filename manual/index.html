1การติดตั้ง

โปรแกรมไม่ได้ผ่านการ sign กับ Apple/Microsoft การเปิดครั้งแรกครั้งเดียวจะมีคำเตือนจากระบบปฏิบัติการ — เป็นเรื่องปกติ ทำตามขั้นตอนด้านล่างแล้วครั้งต่อไปเปิดได้ตามปกติตลอด

macOS (ไฟล์ .dmg)

  1. เปิดไฟล์ .dmg แล้วลากไอคอนโปรแกรมลงโฟลเดอร์ Applicationsรอสักครู่ให้คัดลอกเสร็จ แล้ว eject ตัว dmg ได้เลย
  2. เปิดครั้งแรก: คลิกขวาที่ไอคอน → Open → กด Open ยืนยันถ้าระบบยังไม่ยอมเปิด ให้ไปที่ System Settings → Privacy & Security → เลื่อนลงล่างสุด → กด Open Anyway
  3. ครั้งต่อไปดับเบิลคลิกเปิดได้ปกติ
⚠ เครื่อง Mac ที่รองรับตัวติดตั้งนี้สำหรับ Mac ชิป Apple Silicon (M1 ขึ้นไป, ปี 2020+) — ถ้าใช้ Mac Intel รุ่นเก่า แจ้งทีมงานเพื่อขอตัวติดตั้งเวอร์ชัน Intel

Windows (ไฟล์ Setup .exe)

  1. ดับเบิลคลิกไฟล์ Setup .exe
  2. ถ้าขึ้นจอฟ้า "Windows protected your PC"กด More info แล้วกด Run anyway
  3. โปรแกรมติดตั้งอัตโนมัติและเปิดขึ้นมาเองครั้งต่อไปเปิดจาก Start Menu

2ตั้งค่าครั้งแรก (สำคัญ — ทำก่อนใช้งาน)

เปิดโปรแกรมแล้วไปที่แท็บ ตั้งค่า (Settings) ทางเมนูซ้าย ค่าทั้งหมดบันทึกอัตโนมัติและจำไว้ในเครื่อง ไม่ต้องกรอกซ้ำ

  1. วาง FAL API Key ในช่อง "API Key 1 (หลัก)"คัดลอก key จากบัญชี fal.ai ของคุณ (รูปแบบ xxxx:xxxx) แล้วกดปุ่ม ทดสอบ — ต้องขึ้น ✓ ใช้ได้ ก่อนถึงจะเจนได้
  2. (ทางเลือก) ใส่ API Key 2 ถ้าต้องการความเร็วสูงสุดFAL จำกัดความเร็วต่อ 1 key ที่ ~30 งานพร้อมกัน — เพิ่ม key ที่สองแล้วเลื่อน concurrency ได้ถึง 60 ระบบจะสลับใช้ 2 key ให้เองอัตโนมัติ (ดูตารางความเร็วข้อ 9)
  3. ตรวจค่าที่แนะนำจำนวนงานพร้อมกัน: 30 (หรือ 50–60 ถ้ามี 2 key) · Quality เริ่มต้น: Low · Auto-retry: 2 ครั้ง · กันเครื่อง sleep: เปิด
  4. Default Prompt 2 ชุดโปรแกรมมี prompt สำเร็จรูป 2 ชุด (ฉากห้องกระจก / ฉากผนัง charcoal) แก้ไขข้อความได้ในหน้านี้ — ชื่อรูปหุ่นและชื่อโฟลเดอร์เสื้อจะถูกแทนลงใน prompt ให้อัตโนมัติตอนเจน
💡 เรื่อง API KeyKey เก็บอยู่ในเครื่องคุณเท่านั้น ไม่ถูกส่งไปที่อื่นนอกจาก FAL.AI โดยตรง · ค่าใช้จ่ายการเจนทั้งหมดเรียกเก็บผ่านบัญชี fal.ai ของคุณตามจริง

3การเจนภาพแบบ Bulk

ไปที่แท็บ สร้างภาพ (Generate) — ทำ 5 ขั้นจากบนลงล่าง แล้วกดปุ่มเดียวจบ

  1. เลือกรูปหุ่น (1 รูป)คลิกกล่อง "รูปหุ่น" แล้วเลือกรูปหุ่นเปล่าจากเครื่อง — ใช้ได้ทั้ง JPG / PNG / WebP และ HEIC จาก iPhone โดยตรง (โปรแกรมแปลงและแก้การหมุนให้เอง) · ระบบจะแสดงขนาดภาพผลลัพธ์ที่จะได้ ซึ่งสัดส่วนเท่ารูปหุ่นเสมอ
  2. เลือกโฟลเดอร์เสื้อผ้ากด เลือกโฟลเดอร์… — รูปเสื้อทั้งโฟลเดอร์จะแสดงเป็นตาราง คลิกที่รูปเพื่อติ๊กออก/ติ๊กเข้ารายตัวได้ (ค่าเริ่มต้นเลือกทุกรูป) · ชุดเดียวมีรูปหน้า+หลัง: ตั้งชื่อไฟล์ลงท้าย _front และ _back (เช่น dress01_front.jpg + dress01_back.jpg) ระบบจับคู่เป็นชุดเดียวให้อัตโนมัติ (มีป้าย "หน้า+หลัง" บนรูป) และตอนเจนมุม Back จะใช้รูปด้านหลังเป็นต้นแบบเอง
  3. เลือกมุมภาพFront (ด้านหน้า — ปกติ) / Back (หมุนหุ่น 180° เห็นหลังชุด) / Side (หันข้าง หน้าชุดเข้ากระจก) — เลือกได้หลายมุมพร้อมกัน 1 มุม = 1 รูปต่อเสื้อ 1 ตัว (คิดค่าเจนแยกรูป)
  4. ตั้งค่าการเจนเลือก Quality (Low ฿0.2 แนะนำสำหรับงานหลัก / High ฿1.5 เมื่อต้องการรายละเอียดผ้าสูงสุด) · เลือกโฟลเดอร์เซฟผลลัพธ์ · เลือก Default Prompt 1 หรือ 2 (หรือกด Custom พิมพ์เองเฉพาะ batch นี้)
  5. ดูราคาประเมินแล้วกด "▶ เริ่มเจนทั้งหมด"แถบล่างสุดจะคำนวณให้ก่อนเสมอ เช่น "48 เสื้อ × 2 มุม × Low = 96 รูป ≈ ฿19.2" — กดแล้วโปรแกรมพาไปหน้าติดตามงานอัตโนมัติ
ชื่อไฟล์ผลลัพธ์รูปถูกเซฟลงโฟลเดอร์ที่เลือกทันทีที่เสร็จทีละใบ ในชื่อ ชื่อไฟล์เสื้อ_มุม_quality.png เช่น dress_001_front_low.png — เปิดดูได้เลยระหว่างที่ batch ยังรันอยู่

4หน้าติดตามงาน (Dashboard)

แสดงสถานะทุกงานแบบเรียลไทม์ — เปิดทิ้งไว้เฉยๆ ได้เลย ระบบอัปเดตเอง

สถานะความหมาย
รอคิวรอถึงรอบส่งเข้าเจน (ระบบจำกัดจำนวนงานพร้อมกันตามที่ตั้งไว้)
อัปโหลด… / กำลังเจน / เซฟไฟล์…งานกำลังทำ — เฉลี่ยใบละ 1–2 นาที
✓ เสร็จเจนเสร็จและเซฟลงเครื่องแล้ว — การ์ดแสดงรูปผลลัพธ์ทันที
✗ เฟลล้มเหลวหลัง auto-retry ครบแล้ว — กดปุ่ม 🔁 Retry รายรูป หรือ "Retry ที่เฟลทั้งหมด" มุมขวาบน

แถบบนแสดง: จำนวนเสร็จ/กำลังทำ/รอ/เฟล · แถบความคืบหน้า · ความเร็วจริง (รูป/นาที) · เวลาโดยประมาณที่เหลือ

ปุ่มบนการ์ดรูปที่เสร็จแล้ว

  • 🎨 Edit — เปิดรูปนั้นในหน้าแก้รูป (Editor) · หรือคลิกที่ตัวรูปได้เลย
  • ↻ High — เจนรูปนั้นซ้ำเป็นคุณภาพ High ฿1.5 ทันที ใช้หุ่น/เสื้อ/มุม/prompt เดิมทุกอย่าง — นี่คือหัวใจของโหมดผสม: เจน Low ทั้งชุด แล้วอัปเกรดเฉพาะรูปที่ต้องการ

ปุ่มที่แถบหัวของหน้า: ⏸ พักงาน หยุดส่งงานใหม่ชั่วคราว (งานที่กำลังเจนอยู่ทำต่อจนจบ) กดอีกครั้งเพื่อทำต่อ · 🗑 ลบ batch ลบ batch ที่เลือกออกจากรายการ — ไฟล์รูปที่เซฟไว้ในเครื่องไม่ถูกลบ (ถ้ายังมีงานค้าง ระบบจะถามยืนยันก่อน เพราะงานค้างจะถูกยกเลิก)

5การแก้รูป — แท็บ "แก้ไขภาพ (Editor)"

กด 🎨 Edit จากรูปที่เสร็จใน Dashboard — ระบบจำให้เองว่ารูปนี้เจนมาจากเสื้อตัวไหน ทุกการเจนใหม่จะแนบรูปเสื้อต้นฉบับไปเทียบอัตโนมัติ ไม่ต้องอัปโหลดซ้ำ

เครื่องมือประเภทวิธีใช้
① ความสั้น–ยาวของชุดเจนใหม่ · ฿1.5เลือก "สั้นลง" หรือ "ยาวขึ้น" → ลาก slider 10–50% (ทีละ 5%) → กด Generate · % เป็นค่าประมาณจาก AI — ระดับใกล้กัน (ห่าง 5%) ผลอาจใกล้เคียงกัน ถ้าไม่ถูกใจกดเจนซ้ำได้
② Match สีให้ตรงต้นฉบับเจนใหม่ · ฿1.5กดปุ่มเดียว — ใช้เมื่อรู้สึกว่าสี/ลายของผลเจนเพี้ยนจากเสื้อจริง ระบบส่งรูปต้นฉบับไปเทียบและแก้ให้โดยไม่เปลี่ยนทรง
③ Dull ⟷ Vibrantทันที · ฟรีลาก slider ลด/เพิ่มความสดของสี (−50 ถึง +50) เห็นผลทันทีบนรูป
④ สว่าง ⟷ มืด (Exposure)ทันที · ฟรีปรับความสว่างแบบปุ่ม Exposure ใน iPhone Photos ใช้คู่กับข้อ ③ ได้
  • เครื่องมือ ①–② ส่งงานเข้าคิวใน Dashboard — ผลออกเป็นไฟล์ใหม่ (ต่อท้าย _edit) ไม่ทับของเดิม
  • เครื่องมือ ③–④ เมื่อปรับแล้วกด 💾 บันทึกเป็นไฟล์ใหม่ — ได้ไฟล์ _adj เพิ่ม ต้นฉบับอยู่ครบ
  • แถบ "เวอร์ชัน" ใต้รูปใช้สลับดูต้นฉบับ/ผลแก้ไขทุกรุ่นของรูปนั้น

6ระบบกู้งานอัตโนมัติ — ปิดเครื่อง/ไฟดับ ไม่มีปัญหา

  • ทุกงานบันทึกลงฐานข้อมูลในเครื่องตลอดเวลา — ปิดโปรแกรม, เครื่องดับ, เน็ตหลุด แล้วเปิดใหม่ = ทำต่อจากจุดเดิมอัตโนมัติ
  • งานที่ส่งเข้าระบบ AI ไปแล้วตอนเกิดเหตุ จะถูกติดตามต่อจากงานเดิม — ไม่เจนซ้ำ ไม่เสียเงินซ้ำ
  • รูปที่เสร็จแล้วไม่ถูกแตะต้อง ระบบทำเฉพาะส่วนที่ค้าง (ทดสอบจริงแล้ว: ปิดโปรแกรมกลาง batch 50 รูป เปิดใหม่จบครบ 50/50)
  • ระหว่างมีงานรัน โปรแกรมกันเครื่อง sleep ให้อัตโนมัติ — รันข้ามคืนได้ แค่เสียบปลั๊กและต่อเน็ตทิ้งไว้

7เคล็ดลับการใช้งานจริง

  • โหมดผสมประหยัดสุด: เจนทั้งชุดด้วย Low ฿0.2 → ไล่ดูผล → กด ↻ High เฉพาะรูปที่จะใช้จริงหรือรายละเอียดยังไม่พอ — ที่ 10,000 รูป ต่างกัน ฿1,800 vs ฿15,000
  • งานใหญ่ให้รันข้ามคืน: ตั้ง batch ก่อนเลิกงาน เช้ามาเก็บผล — ที่ความเร็วจริง ~12–20 รูป/นาที ได้คืนละ 8,000–15,000 รูป
  • รูปหุ่นถ่ายครั้งเดียวใช้ตลอด: ระบบอัปโหลดรูปหุ่นครั้งเดียวต่อ batch — ใช้รูปหุ่นเดิมทุก batch จะได้ภาพชุดที่ฉากสม่ำเสมอทั้งแคตตาล็อก
  • ตั้งชื่อไฟล์เสื้อให้ดี: ชื่อไฟล์เสื้อ = ชื่อไฟล์ผลลัพธ์ ใช้รหัสสินค้าเป็นชื่อไฟล์จะจับคู่กลับง่ายที่สุด
  • ถ่ายจาก iPhone ได้เลย: ไฟล์ HEIC และรูปที่ถ่ายแนวไหนก็ตาม โปรแกรมแปลงและจัดการการหมุนให้อัตโนมัติ
  • เจนซ้ำ = สุ่มใหม่: AI ให้ผลไม่ซ้ำกันทุกครั้ง — รูปที่เฟลกด 🔁 Retry ได้เลย ส่วนรูปที่เสร็จแล้วแต่ไม่ถูกใจ (สีเพี้ยน ท่าแปลก) ใช้ปุ่ม ↻ High (฿1.5) เจนใหม่แบบคมสุด หรือเครื่องมือแก้รูปใน Editor — คิดค่าใช้จ่ายเฉพาะรูปนั้นรูปเดียว

8การแก้ปัญหา

อาการวิธีแก้
กดเริ่มเจนไม่ได้ (ปุ่มเทา)เช็คว่าครบ 4 อย่าง: รูปหุ่น · เลือกเสื้ออย่างน้อย 1 · เลือกมุมอย่างน้อย 1 · เลือกโฟลเดอร์เซฟผล และใส่ API Key แล้วในหน้าตั้งค่า
ทดสอบ Key ขึ้น ✗ ใช้ไม่ได้เช็คว่าคัดลอก key ครบทั้งเส้น (มีเครื่องหมาย : คั่นกลาง) · เช็คยอดเงิน/สถานะบัญชีที่ fal.ai
งานเฟลหลายรูปติดกันมักเป็นปัญหาเน็ตหรือเครดิต FAL หมด — เช็คเน็ต เช็คยอดที่ fal.ai แล้วกด "Retry ที่เฟลทั้งหมด"
ช้ากว่าที่คาดเวลาเจนของ AI แกว่งตามช่วงเวลา (1–2 นาที/รูป) · เพิ่ม concurrency ในตั้งค่า (สูงสุด 30 ต่อ 1 key) · เพิ่ม API Key 2
สีผลลัพธ์เพี้ยนจากเสื้อจริงเปิดรูปใน Editor → กด Match Color (฿1.5) — ระบบเทียบกับรูปต้นฉบับให้อัตโนมัติ
รูปหุ่น/ผลลัพธ์เอียงหรือกลับหัวไม่ควรเกิด (ระบบแก้ EXIF ให้แล้ว) — ถ้าเจอ ให้เปิดรูปต้นทางในโปรแกรมดูรูปแล้ว export เป็น JPG ใหม่ก่อนใช้ และแจ้งทีมงาน
อยากรู้ว่าไฟล์ผลอยู่ไหนโฟลเดอร์ที่เลือกไว้ในขั้นตอนที่ 4 ของหน้าสร้างภาพ — ไฟล์เข้าทันทีที่แต่ละรูปเสร็จ
