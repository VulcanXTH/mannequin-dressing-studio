# คู่มือติดตั้ง Mannequin Dressing Studio

โปรแกรมไม่ได้ผ่านการ sign (ตามที่ตกลงกัน) การเปิด**ครั้งแรก**จะมีคำเตือนจากระบบ — ทำตามนี้ครั้งเดียว หลังจากนั้นเปิดได้ปกติตลอด

## macOS (ไฟล์ .dmg — สำหรับ Mac ชิป Apple Silicon: M1 ขึ้นไป)

1. เปิดไฟล์ `Mannequin Dressing Studio-x.x.x-arm64.dmg` แล้วลากไอคอนโปรแกรมไปที่โฟลเดอร์ Applications
2. เปิดครั้งแรก: **คลิกขวาที่ไอคอนโปรแกรม → Open → กดปุ่ม Open** ในกล่องคำเตือน
   - ถ้าขึ้นว่า "ไม่สามารถเปิดได้" ให้ไปที่ System Settings → Privacy & Security → เลื่อนลงล่าง → กด **Open Anyway**
3. เปิดครั้งต่อไป ดับเบิลคลิกได้ปกติ

## Windows (ไฟล์ Setup .exe — Windows 10/11 64-bit)

1. ดับเบิลคลิก `Mannequin Dressing Studio Setup x.x.x.exe`
2. ถ้าขึ้นหน้าจอฟ้า "Windows protected your PC" → กด **More info** → กด **Run anyway**
3. โปรแกรมติดตั้งเองอัตโนมัติและเปิดขึ้นมา — ครั้งต่อไปเปิดจาก Start Menu ได้เลย

## เริ่มใช้งานครั้งแรก

1. ไปที่แท็บ **ตั้งค่า (Settings)** → วาง FAL API Key → กด **ทดสอบ** ให้ขึ้น ✓
2. (แนะนำ) ถ้าต้องการความเร็ว 20+ รูป/นาที ให้ใส่ API Key 2 แล้วเลื่อน concurrency เป็น 50–60
3. ไปแท็บ **สร้างภาพ (Generate)**: เลือกรูปหุ่น 1 รูป → เลือกโฟลเดอร์เสื้อ → เลือกมุมภาพ → เลือก Quality (แนะนำ Low ฿0.2 แล้วค่อยกด ↻ High เฉพาะรูปที่ต้องการ) → เลือกโฟลเดอร์เซฟผล → กดเริ่ม
4. ตามงานที่แท็บ **ติดตามงาน (Dashboard)** — ปิดโปรแกรม/เน็ตหลุดได้ เปิดใหม่ระบบทำต่อเอง

หมายเหตุ: รูปจาก iPhone (.heic) ใช้ได้เลย โปรแกรมแปลงและแก้การหมุนให้อัตโนมัติ
